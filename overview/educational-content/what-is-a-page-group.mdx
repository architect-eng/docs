title: "What is a Page Group?"
description: ""
---

A **Page Group** is a set of pages that all work together toward the same goal, like a campaign.

You can think of it as a “home” for all the different versions of a page that share the same topic, audience, or objective.

![You can switch between your page groups or create a new one at the tab on the top. ](/overview/images/271ecad7-2f6d-4924-b75f-137bcd45a3cf.png)

You can switch between your page groups or create a new one at the tab on the top. 

### Page Group Settings

Page Group Settings let you control **how those related pages behave together** , things like shared context, shared knowledge/product configs, and any defaults that apply across all pages in the group.
