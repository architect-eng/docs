---
title: "How to find Hubspot Webhook URL for Enrichment"
description: "Create a HubSpot webhook URL you can share with Architect for enrichment workflows"
---

You should be able add the form responses into your Hubspot contacts list through the Automation Workflow feature in Hubspot.

**Step 1: (Hubspot) Create a New Workflow**

- Navigate to Automations → Workflows
- Create a new workflow from scratch
- Select Custom trigger type

**Step 2:  (Hubspot) Configure External Event Trigger**

- Choose "Receive a webhook from an external app" from the custom events and external events options
- Click "Add a webhook"
- Name the webhook (e.g., "Architect")
- Click Continue
- Copy the generated webhook URL

**Step 3: Send Architect Webhook URL**  

- share the HubSpot webhook URL (copied in Step 2) with your point of contact on the Architect team.
- Wait for their confirmation that the webhook destination has been set up and is live.

**Step 4: Test**

- **Trigger a Test:** Architect will send a test trigger as soon as this is done.
- **Map Properties & Build Actions:** Map the incoming data (like the test email) to standard HubSpot contact properties. Add an action to your workflow (like "Create or update contact") using those mapped properties.
- **Publish:** Review and turn the HubSpot workflow **ON**.
