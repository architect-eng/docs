# **Setting Up an A/B Test**

---

### **Step 1: Select Your Campaign**

- Log into Google Ads.
- From the left-hand navigation, click **Campaigns**.
- Select the **campaign** where you want to run the test.
- Switch to the **Ads & Extensions tab** so you’re looking at the ad-level view (not just campaign or ad group).

---

### **Step 2: Duplicate Your Existing Ad**

- Locate the ad you want to test (Control ad).
- Check the box next to it and select **Copy → Paste** (or use “Duplicate” if available in the UI).
- This ensures targeting, bidding, and creative remain identical.

---

### **Step 3: Edit the Test Ad**

- Open the duplicated ad.
- Update the **Final URL** (or Tracking Template if you use that for UTMs) to the **Architect landing page URL**.
- Keep everything else — headlines, descriptions, paths, extensions — exactly the same.
- Save the ad.

---

### **Step 4: Analyze Results**

- Run both ads simultaneously for a consistent period (2–4 weeks minimum).
- Review key KPIs:
    - **CTR (Click-through Rate)** → Does one page attract more clicks?
    - **CPC (Cost per Click)** → Is traffic to one page cheaper?
    - **Conversion Rate** → Does one page drive more leads/sales?
- Use these insights to determine whether the Architect landing page outperforms your existing one.

---

✅ **Tips for Customers:**

- Keep only the landing page as the variable — don’t change copy, targeting, or extensions.
- Let both ads run with equal weighting; don’t pause one mid-test.
- Use at least a couple weeks of data (longer if volume is low) to reach statistical confidence.
