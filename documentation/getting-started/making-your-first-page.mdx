## Create a Page Group

1. Go to Page Groups.
2. Click **+ New Page Group**.
3. Fill in the relevant information and create the Page Group.

## Watch the agent build

When you create a page, the agent generates the layout, components, and copy.

If you want to change something, you can ask the agent to edit the page and iterate until it looks right.

## Publish and share

1. Go to the page.
2. Click **Publish**.
3. Copy the page link.

Your page link is hosted on an Architect domain by default. You can set up your own domain later.
