title: "How to Set up Domain?"
description: ""
---

1. Go to “Domains” tab in Architect’s Platform 
2. Click on **+ Add Domain** and add the domain you want to run our pages on 

**Note:** do not include “www” before your domain.

![Screenshot 2025-10-27 at 17.43.17.png](/documentation/images/Screenshot_2025-10-27_at_17.43.17.png)

1. Set up your DNS Records as instructed in the web page
2. Click **“Check Now”** in the top right once done

You should see that your domain status is **verified** if all is set up correctly.
